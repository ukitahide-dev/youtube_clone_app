# youtube_clone_app

YouTube風のWebアプリ（React + Django）

## 使用技術
- Frontend: React / Vite
- Backend: Django / Django REST Framework
- Auth: JWT (SimpleJWT)
- DB: SQLite（開発環境）

## 主な機能
- ユーザー登録・ログイン・ログアウト
- 動画投稿・再生・編集・削除・いいね・わるいね・コメント・検索
- 視聴用/投稿用プレイリスト作成・編集・削除
- チャンネル登録・解除


## 開発背景
Webアプリ開発の一連の流れ（認証・API設計・フロント連携）を学ぶために制作。
